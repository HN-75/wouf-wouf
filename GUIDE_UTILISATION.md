# Guide d'Utilisation - Wouf Wouf

## Installation

1. Télécharge le fichier `wouf_wouf.apk`
2. Sur ton téléphone Android, active "Sources inconnues" dans les paramètres
3. Installe l'APK
4. Accorde les permissions (micro, Bluetooth si besoin)

## Premier lancement

1. **Choisis ton genre** : L'app adapte les phrases ("maître", "maîtresse", ou "humain")
2. **Donne un nom à ton chien** (optionnel) : Pour personnaliser les messages
3. C'est prêt !

## Utilisation

### Mode Traduction
1. Appuie sur le gros bouton au centre
2. Ton chien aboie
3. Relâche le bouton
4. L'app analyse et traduit l'aboiement
5. La phrase s'affiche et est lue à voix haute

### Mode Apprentissage
1. Va dans "Apprentissage" depuis le menu
2. Enregistre un aboiement de ton chien
3. Dis à l'app ce que ça signifie (Faim, Jouer, Peur, etc.)
4. Plus tu fais ça, plus l'app devient précise pour TON chien

### Bluetooth (optionnel)
1. Va dans "Bluetooth" depuis le menu
2. Connecte un micro externe ou collier Bluetooth LE
3. L'app utilisera ce micro au lieu de celui du téléphone

## Les 6 émotions détectées

| Émotion | Ce que ça veut dire |
|---------|---------------------|
| **Faim** | Ton chien a faim ou soif |
| **Jouer** | Il veut jouer, s'amuser |
| **Peur** | Il a peur de quelque chose |
| **Sortir** | Il veut aller dehors (pipi, balade) |
| **Douleur** | Quelque chose lui fait mal |
| **Joie** | Il est super content |

## Conseils

- **Entraîne l'app** : Plus tu utilises le mode apprentissage, plus c'est précis
- **Contexte** : L'heure et les habitudes aident (aboiement le matin = sortir, le soir = faim)
- **Patience** : Au début c'est approximatif, ça s'améliore avec le temps

## Problèmes courants

**L'app ne détecte pas d'aboiement**
- Rapproche le téléphone de ton chien
- Vérifie que le micro fonctionne
- L'aboiement doit être assez fort

**La traduction semble random**
- Utilise le mode apprentissage pour améliorer
- L'app apprend les patterns de TON chien

**Pas de son**
- Vérifie le volume du téléphone
- Vérifie que le TTS français est installé
